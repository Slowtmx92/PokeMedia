# PokeMedia
PokéMedia : L'encyclopédie Pokémon complète et intuitive. > * Base de données : 1025 Pokémon (Générations 1 à 9).  Filtres intelligents : Recherche par région, par forme spéciale ou par ordre alphabétique.  Fiches détaillées : Descriptions officielles, statistiques et types. Divertissement : Module de quiz intégré pour tester tes réflexes d'expert.
